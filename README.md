# Apple Site Clone

A pixel-perfect Apple.com clone built with React + TypeScript, featuring:

- 11 fully designed pages (Home, Store, Mac, iPhone, iPad, Watch, AirPods, TV & Home, Entertainment, Accessories, Support)
- SVG product illustrations with detailed device renderings
- Scroll-triggered animations, parallax effects, and cinematic text reveals
- Interactive product configurators with live pricing
- Apple-style comparison tables for all product lines
- Step-by-step buy forms with progress tracking
- Canvas particle systems, gradient mesh backgrounds
- Magnetic buttons, marquee scrollers, comparison sliders
- Responsive navigation with mobile hamburger menu
- Scroll progress ring indicator

Built as a single-file HTML artifact — open `index.html` to view.
